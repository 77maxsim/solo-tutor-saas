Parent list state (open modal on row click)

In the students list component (e.g., client/src/pages/students/students-table.tsx or similar), add state and renderer:

// imports
import { useState } from "react";
import { EditStudentModal } from "@/components/modals/edit-student-modal"; // adjust path if different
import type { Student } from "@/types"; // adjust path if needed

// inside component
const [editingStudent, setEditingStudent] = useState<Student | null>(null);

// near return(), render modal once
{editingStudent && (
  <EditStudentModal
    open={!!editingStudent}           // or isOpen / onOpenChange depending on your API
    student={editingStudent}
    onClose={() => setEditingStudent(null)}
  />
)}


If the modal uses isOpen / onOpenChange, adapt prop names accordingly (no other logic changes).

Make the entire row clickable

In the row component (e.g., StudentRow.tsx or within map of rows), wrap the row container (<tr> for table rows or <div> for cards) with an onClick that opens the modal:

// row container
<tr
  onClick={() => setEditingStudent(student)}
  role="button"
  tabIndex={0}
  onKeyDown={(e) => {
    if (e.key === "Enter" || e.key === " ") {
      e.preventDefault();
      setEditingStudent(student);
    }
  }}
  className="hover:bg-muted cursor-pointer"
>
  {/* cells... */}
</tr>


If your list uses <div> instead of <tr>, apply the same handlers to that element.
